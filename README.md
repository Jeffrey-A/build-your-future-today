## Running Dev Version

- `git clone` https://github.com/Jeffrey-A/build-your-future-today.git
- `cd /build-your-future-today`
- `npm i`
- `node seed.js`
- `cd /front-end`
- `npm i`
- `npm run build`
- `cd ../`
- `node index.js`
- Go to: http://localhost:5000/
